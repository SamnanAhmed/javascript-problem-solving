# JavaScript Problem Solving

A collection of JavaScript problem-solving exercises, logic challenges, 
and mini-projects built while strengthening core JS fundamentals.

## 📁 Contents

- **Problem Solving/** — Daily coding challenges covering logic, arrays, 
  objects, and ES6+ concepts
- **TODO List/** — A functional To-Do List app demonstrating practical 
  use of `map()`, `filter()`, and `reduce()` for managing tasks

## 🛠️ Tech Stack

- JavaScript (ES6+)
- HTML5 / CSS3

## 🎯 Purpose

Built as part of a structured JavaScript learning path to strengthen 
problem-solving skills and practical application of array methods, 
DOM manipulation, and core JS concepts.
